# New chat — paste files in order

When starting a new ChatGPT session, paste the contents of these files **in this order** into the first message (or system/context area):

1. **workflow/STATE.md** — single source of truth for current state
2. **workflow/RULES.md** — role split and invariants
3. **workflow/SESSION_HEADER.md** — session rules, current task, objective
4. **workflow/PROJECT_CONTEXT.md** — project anchor (if not already covered by STATE)
5. **workflow/DECISIONS_LOG.md** — recent decisions (last N entries)
6. **workflow/TASK_QUEUE.md** — active and next tasks

Then add your actual request or question.
