# New chat — paste files in order

When starting a new ChatGPT session, paste the contents of these files **in this exact order** into the first message:

a) workflow/BOOTSTRAP_PROMPT.md  
b) workflow/STATE.md  
c) workflow/RULES.md  
d) workflow/SESSION_HEADER.md  
e) workflow/PROJECT_CONTEXT.md  
f) workflow/TASK_QUEUE.md  
g) workflow/DECISIONS_LOG.md (only if there were decisions)  
h) workflow/VERIFY.md (optional, but recommended)

Then add this line:

**Resume the project. Propose the next step. Respect invariants. No assumptions.**

---

Short version (minimal context):

1. workflow/STATE.md  
2. workflow/RULES.md  
3. workflow/SESSION_HEADER.md  
4. Your message
